# ClaimGuard - AI-Powered Insurance Policy Analysis

![ClaimGuard Logo](https://via.placeholder.com/800x200/667eea/ffffff?text=ClaimGuard+AI)

ClaimGuard is an advanced AI-powered platform that analyzes insurance policy documents to provide comprehensive risk assessment, key insights, and personalized recommendations. It helps users understand complex insurance policies before making claims, ensuring they are fully aware of coverage, exclusions, and potential out-of-pocket expenses.

### üéØ Key Features

- **Smart Policy Analysis**: Upload PDF insurance policies and get instant AI-powered analysis
- **Risk Assessment**: ML-based risk prediction for coverage, costs, and claim delays
- **Claim Simulator**: Simulate claims to understand out-of-pocket expenses
- **PDF Report Generation**: Download detailed PDF reports with visualizations
- **Policy Comparison**: Compare multiple policies side-by-side
- **User Dashboard**: Track all analyzed policies with statistics
- **OTP-based Authentication**: Secure login with phone verification

### Frontend
- **HTML5/CSS3**: Responsive design with modern CSS features

- **OTP Verification**: Phone-based authentication
